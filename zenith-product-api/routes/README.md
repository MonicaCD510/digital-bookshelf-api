# Zenith Product API

A RESTful API built with Node.js, Express, MongoDB, and Mongoose for managing product inventory.

## Features

- Create products
- Read all products
- Read single products
- Update products
- Delete products
- Filtering
- Sorting
- Pagination

## Technologies Used

- Node.js
- Express
- MongoDB Atlas
- Mongoose
- dotenv
- Nodemon

## API Endpoints

### Create Product
POST /api/products

### Get All Products
GET /api/products

### Get Single Product
GET /api/products/:id

### Update Product
PUT /api/products/:id

### Delete Product
DELETE /api/products/:id
